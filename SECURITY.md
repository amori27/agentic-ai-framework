# Security Policy

## Supported Versions

We currently support the latest release with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue. Instead, email us directly at [amori27@users.noreply.github.com](mailto:amori27@users.noreply.github.com).

We will respond within 48 hours and work with you to address the issue promptly.
